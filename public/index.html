Visualiser Character Studio

Character art → directional views → game sprites

Step 1

Add a character reference

Upload an illustration, rendered model view, or GLB model. The description box is used to explain hidden details and the exact pose you want. Advanced controls can now lock cloak, cape, mantle, and draped-garment structure.

Checking renderer…

GLB and glTF load directly. ZIP uploads are unpacked in the browser; if a DAE is found it is converted to GLB automatically, even when the texture folders are messy. Then rotate the model and capture each reference.

Advanced generation options

For capes, cloaks, mantles, detached sleeve drapes, robe trains, or similar layered garments, set the structure here. The renderer also sends upper and lower detail crops so shoulder attachments, drape shapes, and hems are easier to preserve.

Step 2
